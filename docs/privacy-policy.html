Politica de privacidad de RecuerdaMed

Ultima actualizacion: 2026-04-21

1. Datos que recopilamos

  • Correo electronico para autenticacion.
  • Nombre, apellido, fecha de nacimiento y genero para el perfil.
  • Medicamentos, horarios y estado de toma.
  • Datos tecnicos minimos necesarios para operar el servicio.

2. Como usamos los datos

  • Crear y mantener tu cuenta.
  • Sincronizar tus registros entre dispositivos.
  • Programar recordatorios y mostrar resumenes diarios.
  • Proteger la seguridad basica del servicio.

3. Comparticion de datos

No vendemos tus datos personales. Los datos se procesan con proveedores de infraestructura necesarios para autenticacion, almacenamiento y funcionamiento de la aplicacion.

4. Retencion y control

Conservamos los datos mientras tu cuenta este activa o mientras sean necesarios para operar el servicio. Puedes solicitar o ejecutar la eliminacion de cuenta siguiendo la pagina de borrado de cuenta.
