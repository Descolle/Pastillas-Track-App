# Custom Notification Sounds

## Sound Files for Louder Notifications

This folder contains custom notification sound files for RecuerdaMed to ensure medication reminders are loud enough.

## Required Files:

1. **pill_reminder_1.wav** - Loud pill reminder sound
2. **pill_reminder_2.wav** - Alternative loud reminder
3. **pill_reminder_3.wav** - Another loud option

## How to Add Sounds:

1. Add your .wav or .mp3 sound files to this folder
2. Make sure they are:
   - **Format**: WAV or MP3
   - **Duration**: 1-3 seconds
   - **Volume**: High quality, loud output
   - **Size**: Under 100KB each

## Sound Sources:

You can download free notification sounds from:
- Zapsplat
- Freesound
- Mixkit
- Or create your own

## Naming Convention:

Use the exact filenames:
- pill_reminder_1.wav
- pill_reminder_2.wav  
- pill_reminder_3.wav

The app will randomly select one of these sounds for each notification to provide variety.
